# 🔢 Математикийн суурь чадвар шалгах (Эерэг ба Сөрөг тоо)
* Энэхүү төсөл нь суралцагчдын эерэг болон сөрөг тооны үйлдлийн суурь мэдлэгийг сэргээн бататгах, өөрийгөө үнэлэх боломжийг олгох зорилготой интерактив вэб аппликейшн юм.

## 🎯 Зорилго
* Суралцагчид математикийн хамгийн чухал суурь болох рациональ тооны үйлдлүүдийг (нэмэх, хасах, үржих, хуваах) хөгжилтэй бөгөөд хялбар аргаар бататгах.

## 🚀 Гол онцлогууд
* 3 түвшний дасгал: Нийт 60 бодлогыг хүндрэлийн зэрэглэлээр (Дунд, Ахисан, Мастер) ангилсан.

* Шуурхай хариу үйлдэл: Хариуг оруулаад "Шалгах" товч дарахад зөв хариулт 🟢 ногоон, буруу хариулт 🔴 улаан өнгөөр тэмдэглэгдэнэ.

* Онооны систем: Нийт 60 бодлогоос хэдийг нь зөв бодсоныг бодит хугацаанд тооцож харуулна.

* Интерактив загвар: Tailwind CSS ашиглан хийгдсэн, ашиглахад хялбар орчин үеийн загвартай.
## 1. Нэмэх ба хасах үйлдэл (+ ба -)
* Дүрэм: "Тэмдэг ижил байвал нэмнэ, тэмдэг өөр байвал хасна.
* "Ижил тэмдэг: Жишээ нь, $-5 + (-3)$ гэвэл хоёулаа "хасах" тэмдэгтэй байгаа биз? Энэ үед тоонуудыг нь нэмээд (5+3=8), ерөнхий тэмдэгийг нь тавина. Хариу нь $-8$.
* Өөр тэмдэг: Жишээ нь, $-10 + 4$. Нэг нь "хасах", нөгөө нь "нэмэх". Энэ үед их тооноос бага тоог хасаад (10-4=6), их тооныхоо тэмдэгийг тавина. Хариу нь $-6$.
## 2. Үржих ба хуваах үйлдэл (* ба :)
* Дүрэм: "Тэмдэг ижил байвал нэмэх, тэмдэг өөр байвал хасах болно.
* "Ижил тэмдэг: Жишээ нь, $-5 * (-3)$ гэвэл хоёулаа "хасах" тэмдэгтэй байгаа биз? Энэ үед тоонуудыг үржихэд (-5)* (-3)=8).
* Өөр тэмдэг: Жишээ нь, $-10 * 4$. Нэг нь "хасах", нөгөө нь "нэмэх". Энэ үед тоонуудыг үржихэд (-10)*4=-40).
